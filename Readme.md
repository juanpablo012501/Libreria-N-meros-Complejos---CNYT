Libreria de números complejos.

Este pequeño proyecto busca reforzar el entendimiento de los
números complejos para su uso luego en los temas de la computación
cuántica.

Para utilizar estas funciones, ha de ingresar tuplas.
La primera ordeada es la parte real y la segunda es la
parte imaginaria del número complejo. Evidentemente, dependiendo
de la naturaleza la operación (unaria o binaria) usted ha de
emplear la cantidad apropiada de números complejos.

Las operaciones disponibles son:
adición, sustracción, multiplicación, división, determinar
el modulo y conjugado del número, las conversiones de representación
cartesiana a polar y vicesversa; y determinar la fase del número.

Por: Juan Pablo Vélez Muñoz.
